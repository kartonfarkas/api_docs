Listing Combined Segments
=========================

Generates a list of combined segments, which are used for filtering according to different criteria.

Endpoint
--------

``GET /api/v2/segments/combined``

Result Example
--------------

.. code-block:: json

   {
      "replyCode": 0,
      "replyText": "OK",
      "data": [
         {
            "id": 33,
            "name": "test_segment"
         }
      ]
   }
